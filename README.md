# Context App Website

This repository hosts the marketing site for **Context**, a native Apple‑ecosystem productivity app.

## Live Site

GitHub Pages: https://timmetro69.github.io/mycontextapp.com-web-site/

## Pages

- `index.html` — Home
- `feature-location.html` — Location‑Aware Orchestration
- `feature-nlp.html` — Natural‑Language Capture
- `feature-privacy.html` — Privacy‑First
- `feature-ecosystem.html` — Apple Ecosystem
- `about.html` — About
- `contact.html` — Contact (placeholder form)

## Assets

- `context-app-icon.png` — app icon used across the site
- `style.css` — shared styles

## Development

This is a static HTML/CSS site. Edit the files directly and push to `main`.

## GitHub Pages

Pages is enabled on `main` with root `/`.
